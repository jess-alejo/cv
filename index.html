About Me

Hi! I'm a Senior Developer at Document Intelligence Solutions OPC. I maintain desktop applications I've created with C# and WPF and collaborate to improve our Ruby on Rails back-end applications. I'm currently learning Advanced JavaScript and Modern Front-End Development.

Whatever our profession is, we need tools that facilitate our work and be productive. Having the right tools helps solve problems efficiently and makes us comfortable. Here are some of my favorite tools when developing:

Desktop

C#

C# is my favorite programming language. I successfully built an Accounting System in 2014, which solved most of the problems the company had.

WPF

My passion coding in C# introduces me to Windows Presentation Foundation which helps me create wonderful user interfaces in Windows-based applications.

MySQL

MySQL is my prepared database when creating desktop applications. I find it easy to use and learn, and resources are easily accessible.

Front-end

HTML 5

HTML is one of the languages of the web. Many applications are now moving online, and HTML is a necessary tool to have.

CSS 3

CSS makes the web UI beautiful and unique. It's a great tool to have an engaging website and express artistry.

JavaScript

JavaScript is a programming language now used in both client-side and server-side. It makes the web interactive and dynamic.

Back-end

Ruby

I like Ruby because of the semantics and metaprogramming concepts. I met Yukihiro 'Matz' Matsumoto, the creator of Ruby, in RubyConfPH 2017.

Ruby on Rails

I spent most of my web development in Ruby on Rails. Personally, it is extensive, and the learning curve is steep.

PostgreSQL

Most of the Rails application we've been developing uses PostgreSQL, but I rarely touch them internally. It is stable and reliable, especially in handling massive data.
